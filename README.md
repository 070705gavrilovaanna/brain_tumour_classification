# Классификация опухолей мозга на МРТ-снимках

Проект по классификации медицинских изображений: определение наличия опухоли на МРТ-снимках мозга. Сравниваются три архитектуры: самописная CNN, CNN с Dropout и предобученная ResNet18.

**Ссылка на проект:** [brain_tumor_classification.ipynb](https://colab.research.google.com/drive/1qfWfnHe19-XqQ3OeSIunqcTA2nIRwRaI#scrollTo=zUiuqRGfTUeL)

---

## Этапы работы

**Технологии:** Python, PyTorch, Torchvision, Scikit-learn, NumPy, Matplotlib, Seaborn

### 1. **Загрузка и подготовка данных**
   - Распаковка датасета с изображениями (с опухолью / без)
   - Разделение на train/test в пропорции 80/20
   - Визуализация распределения классов

### 2. **Аугментация и предобработка**
   - Изменение размера до 128×128 пикселей
   - Нормализация значений пикселей
   - Аугментация: случайные повороты для увеличения вариативности

### 3. **Моделирование**
   Сравнила 3 архитектуры:
   - **Simple CNN:** 3 сверточных слоя с BatchNorm
   - **CNN + Dropout:** аналогичная архитектура с регуляризацией
   - **ResNet18:** предобученная на ImageNet модель

### 4. **Обучение**
   - 10 эпох для каждой модели
   - Сохранение лучших весов
   - Визуализация динамики loss и accuracy

### 5. **Результаты**
   - **Simple CNN:** 70.6%
   - **CNN + Dropout:** 70.6%
   - **ResNet18:** 88.2%

---

## Сравнение моделей

| Модель | Accuracy | Precision (no/yes) | Recall (no/yes) |
|--------|----------|-------------------|-----------------|
| Simple CNN | 70.6% | 0.67 / 0.72 | 0.50 / 0.84 |
| CNN + Dropout | 70.6% | 0.63 / 0.75 | 0.60 / 0.77 |
| ResNet18 | **88.2%** | **0.89 / 0.88** | **0.80 / 0.94** |

---

## Что я узнала

1. **Transfer Learning работает:** ResNet18 дала прирост +17.6% к точности
2. **Dropout не помог** на малом датасете (253 изображения)
3. **Confusion Matrix показала** что простые модели чаще ошибались в пользу класса "yes"
4. **Recall важнее accuracy** в медицинских задачах - ResNet18 нашла 94% опухолей

**Ключевой вывод:** Предобученные модели значительно эффективнее самописных, особенно на небольших датасетах.
